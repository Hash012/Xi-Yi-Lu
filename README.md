# 《習藝錄》自序

夫CS，世之顯學也。探賾索隱，窮究構造之變；執簡馭繁，竟繫方寸之間。然本科課務，疆域益廣，非復舊時溝洫。夫TCS、系統、編碼，經典深晦，嚼咽維艱；更需兼吞數據科學、AI、工程實踐，萬象森羅，紛至沓來，竟不能尋常遍歷之。昔日『教條格致』之法，難應萬變。

況乎LLM橫空，點石成金，同儕競逐，人心若沸。欲速，而不能速達；不速，則困於槽櫪。何止兩難。懸崖壁立與面前，百獸哀嚎與左右，不退，遂攀焉。此庫所錄，乃途中所參之心法，所目之知識也，冀以得『速爾不危，徐而不困』之法門。

余進學人大，忽焉一年半矣。每欲格物，恆苦無門。學此一端，輒牽涉百端，雖云不必盡通，亦不知需涉獵幾何，深究幾許云爾。界限模糊，千綜萬緒。或入海算沙，或觀燈走馬。緣木求魚，徒然擲筆浩嘆。及得師授，又患心猿，似董子之反調，一日窺園有三。見洛城花好。可憐拙手，不堪折枝。

又方寸烈烈，妄圖倖進。行險圖成，而多致敗績。

當是時，進退失據，狀如俳優。左支右絀，五內如焚。神魂蕩越，幾陷潰決。

古人云：『知所先後，則近道矣』。于兹始悟，兼察前非。由是棄絕紛華，收視反聽。欲究無師自通之法，亦正身心修養之學。去矯情之累，從無我之誠。餘事非本，皆付造化。春風若去，任之而已。

大道多歧，且賴一息尚存；學海無涯，所貴知其所止。

今置此庫，蓋圖鍵錄之便，亦備同學之需。知門可羅雀，如覆瓿之用。倘蒙君子過目，糾其偏頗，示以高見，則幸甚。

是為序。

**時維丙午，立春後四日謹識於蝸室**
